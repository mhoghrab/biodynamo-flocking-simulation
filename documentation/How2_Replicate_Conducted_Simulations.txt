A short instruction is given on how to replicate the conducted simulations.

The external library cnpy has to be installed https://github.com/rogersce/cnpy.

We used the random number mofule of BioDynaMo. It looks like this always creates the exact same sequence of numbers.
Therefore, when choosing the same simulation settings the exact same simulation should be executed as in the thesis.

The default values and default bdm.json are the described values in the beginning of chapter 5. 
But make sure to double check in case and experiment with them.

In the following the simulations of the thesis are listed with a small instruction on how to set them up.
The most important / changed parameters are highlighted.
The data used for our plots is also given in the corresponding output/data_xxx folder alongside the matlab script to plot / visualize them.

- data_free_space: "Figure 10: Plot of the reduced average distance..."
- data_wind: "Figure 14: Wind resistance..."
             "Figure 15: Plot of the average velocity within a flock of 500 drifting agents..."

5.1  Flocking in Free Space: Avg neighbor distance (fig 10)
------------------------------------------------------------------------------------------------------------
"n_boids": 500,
"starting_sphere_radius": 200
"simulation_setup": "free_space_avg_dist"
"max_bound": 50000
"min_bound": -50000 
"c_a_3": 0 / 0.05
"c_y": 0.05
"export_distances": true -> will create a csv in output/data that contains the agents in each row and the avg neighbor distance at each simulation step in the columns

in sim_param.h:  Double3 pos_gamma = {100000, 0, 0};
